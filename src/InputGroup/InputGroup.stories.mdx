# InputGroup

```js
import { InputGroup } from 'fundmental-react-native';
```

An **InputGroup** includes form inputs with other children that allow the user to better understand the information being entered.

## Button child

The **InputGroup** children supports buttons by providing the correct styling. Buttons can be shown with a text label or icon.
